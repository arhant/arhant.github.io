Tennis360

Using official data for Tennis provided by ATP, we calculated the predictability of a player winning the next Grad-Slam on Grass, Clay or Hard court.
https://github.com/arhant/Tennis360

Gaming Culture: Game Popularity Predictor

Implemented Data Science and Machine Learning algorithms like K-means, Decision Trees, Random Forest to analyze over 16K game sales data for popularity based on video game, publisher, platform, genre and predicted which game will continue to be popular in different continents based on the critic values and ratings

360FITNESS

Developed and deployed a web app in the cloud for complete fitness. The Product uses AWS Cognito for login & session, Twilio for SMS notifications & communicating with AWS Lex offline for exercise information. Gym search is based on location and route in-between is done by Google Geocoding, Reverse Geocoding & Directions. Data is stored in AWS DynamoDB to calculate BMI & to customize meal plan. AWS Lex is also integrated with Facebook Chat
https://github.com/arhant/360Fitness

Tweet Trends

The System used Front-End hosted on AWS S3, API Gateway to trigger AWS Lambda functions, which used Tweepy to extract tweets and MonkeyLearn for sentimental analysis. SNS and SQS were used for communication, and tweets were plotted on Google Maps with sentiments using Elasticsearch.
https://github.com/arhant/Tweet_Trend

NYC Crime

Worked on a Big Data Project to analyze and clean the data provided by NYPD for crime records till 2016. The system provided detailed graphs showing crime locations, time and frequency based on which hypothesis could be made, pinpointing all the vulnerable locations and peak hours for crime in NYC, thus reducing the probability of crimes.
https://github.com/arhant/NYC-Crime-Analysis
